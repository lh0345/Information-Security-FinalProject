# Password Strength Checker and Security Awareness Tool

A beginner-friendly Information Security final project that checks password strength in real time, educates users about security risks, and promotes better password practices.

## Features

- **Real-Time Password Strength Meter** — visual progress bar with Weak/Medium/Strong levels
- **Password Requirements Checklist** — live checks for length, uppercase, lowercase, numbers, and special characters
- **Security Feedback Messages** — actionable tips to improve password strength
- **Password Visibility Toggle** — show/hide the password with a click
- **Estimated Crack Time** — educational estimate of how fast a password could be cracked
- **Password Entropy Calculation** — displays entropy in bits for strong passwords
- **Random Strong Password Generator** — generates a 16-character password meeting all criteria
- **Educational Security Tips Section** — explains why strong passwords matter, brute-force attacks, credential stuffing, and MFA
- **Dark Mode UI** — modern, responsive design with smooth animations
- **Fully Client-Side** — all analysis happens locally; no data is ever sent or stored

## Technologies Used

- **HTML5** — semantic structure with accessibility labels
- **CSS3** — custom properties, flexbox, grid, animations, responsive design
- **JavaScript (Vanilla)** — DOM manipulation, event handling, regex validation, entropy calculation

No frameworks, no backend, no databases, no external dependencies.

## How to Run Locally

1. Download or clone the repository
2. Open `index.html` in any modern web browser
3. Start typing a password to see real-time analysis

That's it — no installation, no server required.

## Project Structure

```

```

## Security Concepts Demonstrated

- Password strength analysis and scoring
- Authentication fundamentals
- Brute-force attack awareness
- Credential stuffing risks
- Entropy as a measure of password unpredictability
- Defense in depth (passwords + MFA)
